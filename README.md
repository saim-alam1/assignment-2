### **(1) What is JSX, and why is it used?**

**_Ans:_**  
JSX Means JavaScript XML. It allows HTML like markup in JavaScript Code to increase the readability of the code.

### **(2) What is the difference between State and Props?**

**_Ans:_**  
In React state is managed internally by the component to store the data on the other hand Props are the data passed from parent component to the child components.

---

### **(3) What is the useState hook, and how does it work?**

**_Ans:_**  
useState is a builtin function in React it remembers data across every re-renders. It is also very flexible while changing its values.

---

### **(4) How can you share state between components in React?**

**_Ans:_**  
In React, there is many ways to share the states between components using lifting state up for simple cases, by props drilling and also by context APi's.

### **(5) How is event handling done in React?**

**_Ans:_**  
In React, event handlings are used to perform according to users need like onClick, onSubmit, onType etc to handel user interaction event handlers are used. Event handlers in react are written in camel Case.
